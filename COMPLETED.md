## 📋 快速测试步骤

### 1. 在浏览器中测试（推荐先试这个）

打开浏览器访问：**http://localhost:5173/**

**测试单张识别：**
1. 点击"单张识别"标签
2. 点击上传区域选择一张图片
3. 等待图片预览显示
4. 点击"开始识别"按钮
5. 观察进度条和识别结果

**测试批量识别：**
1. 切换到"批量识别"标签
2. 添加多张图片（按住Ctrl多选）
3. 点击"开始批量识别"
4. 查看每个图片的处理状态
5. 完成后点击"导出结果"

### 2. 启动 Electron 桌面应用

**注意**: 由于之前安装依赖时有文件锁定问题，建议：

```bash
# 方法1: 重启后运行（推荐）
# 关闭所有终端和可能的 Electron 进程，然后：
npm run electron:dev

# 方法2: 如果还有问题，先清理
# 删除 node_modules 后重新安装
rm -rf node_modules
npm install
npm run electron:dev
```

## 💡 首次使用提示

### ⚠️ 重要提醒

1. **首次识别会较慢**
   - 需要下载语言包（10-20MB）
   - 请耐心等待，只需一次
   - 之后会使用缓存，速度提升10倍+

2. **如何获得最佳效果**
   - 使用清晰的图片
   - 已知语言时手动选择（更快）
   - 批量处理时分批进行（20-30张）

3. **支持的图片格式**
   - PNG（推荐，无损）
   - JPG/JPEG
   - GIF
   - BMP
   - WebP

## 🔍 故障排查

### 问题1: 页面空白
**解决**: 
- 检查控制台是否有错误
- 确认 Vite 服务器正在运行
- 刷新页面（Ctrl+R）

### 问题2: 识别失败
**解决**:
- 检查网络连接（首次需要下载语言包）
- 尝试其他图片
- 查看控制台错误信息
- 手动选择语言而非自动检测

### 问题3: 速度慢
**解决**:
- 首次使用是正常的（下载语言包）
- 降低图片分辨率
- 手动指定语言
- 查看 PERFORMANCE.md 优化建议

### 问题4: Electron 启动失败
**解决**:
```bash
# 清理并重装
rm -rf node_modules
npm cache clean --force
npm install
npm run electron:dev
```
