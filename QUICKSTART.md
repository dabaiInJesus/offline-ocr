# 快速开始指南

## 🎯 5分钟上手

### 第一步：启动应用

```bash
# 开发模式（推荐初次使用）
npm run electron:dev

# 或者先测试 Web 版本
npm run dev
# 然后访问 http://localhost:5173
```

### 第二步：识别第一张图片

1. **选择图片**
   - 点击上传区域
   - 或拖拽图片到窗口

2. **选择语言**（可选）
   - 自动检测：适合未知语言
   - 手动选择：知道语言时更快

3. **开始识别**
   - 点击"开始识别"按钮
   - 等待进度条完成

4. **查看结果**
   - 识别文本显示在下方
   - 点击"复制文本"即可使用

### 第三步：批量处理（可选）

1. 切换到"批量识别"标签
2. 添加多张图片（支持多选）
3. 点击"开始批量识别"
4. 完成后点击"导出结果"

## ⚡ 提速技巧

### 首次使用
- 📥 首次识别需要下载语言包（10-20MB）
- ⏱️ 请耐心等待，只需一次
- 💾 之后会自动缓存，速度提升10倍+

### 提高速度

- ✅ 确保文字方向正确（水平）

## ❓ 常见问题

### Q: 为什么第一次识别很慢？
A: 需要下载语言包（约10-20MB），下载后会自动缓存。

### Q: 支持哪些图片格式？
A: PNG、JPG、JPEG、GIF、BMP 等常见格式。

### Q: 可以识别手写体吗？
A: 当前版本主要针对印刷体，手写体识别率较低。

### Q: 如何获得更好的识别效果？
A: 
- 使用清晰、高分辨率的图片
- 确保光线充足、对比度高
- 文字尽量水平排列
- 选择合适的语言

### Q: 批量处理有数量限制吗？
A: 理论上无限制，但建议每批20-30张以避免内存问题。

### Q: 数据会上传到云端吗？
A: 不会！所有处理都在本地完成，完全离线。
