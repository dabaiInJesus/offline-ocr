## 🔑 关键要点

### 1️⃣ 语言选择（最重要！）
- ✅ **已知语言 → 手动选择**（提升 20-40%）
- ❌ 避免使用"自动检测"（慢且不准）

### 2️⃣ 图像预处理
- ✅ **低质量图片必开**（提升 15-30%）
- ✅ 自动放大 2 倍 + 灰度 + 二值化
- ⚠️ 增加 20% 处理时间

### 3️⃣ PSM 模式
- 📖 书籍文章 → 单列文本
- 📝 标题段落 → 单个文本块
- 🏷️ 标语路牌 → 单行文本
- 📊 表格表单 → 稀疏文本
- ❓ 不确定 → 自动

### 4️⃣ OEM 引擎
- ⚡ **LSTM** → 日常使用（快且准）
- 🎯 **结合模式** → 追求极致准确
- 🔄 **旧版** → 特殊字体/古老文档

---

## 💡 图片质量建议

### 拍摄技巧
- 📐 保持水平（倾斜 < 5°）
- 💡 均匀照明（避免阴影）
- 🎯 对焦清晰（避免模糊）
- 📏 分辨率 ≥ 300 DPI

### 格式选择
- ✅ PNG（无损，最佳）
- ✅ JPG（高质量 > 90%）
- ❌ 避免过度压缩

## ❓ 常见问题速查

### Q: 中文识别率低？
```
✅ 选"中文简体"而非自动
✅ 开启图像预处理
✅ 使用高分辨率图片
✅ 尝试 PSM "单个文本块"
```

### Q: 英文数字混淆？
```
✅ 选"英文"语言
✅ PSM "单行文本"（如为单行）
✅ 开启预处理增强对比
```

### Q: 速度慢？
```
✅ 用 LSTM 引擎（快 3-5 倍）
✅ 手动选语言（跳过检测）
✅ 关闭不必要的预处理
```

### Q: 批量处理优化？
```
✅ 统一语言设置
✅ 用 LSTM 引擎
✅ 分批处理（20-30 张/批）
✅ 关闭实时预览
```
